# ملاحظات المبرمج

حديقة معرفة شخصية — ملاحظات ودروس في البرمجة والأمن السيبراني باللغة العربية.

[**arab-dev-note.vercel.app**](https://arab-dev-note.vercel.app)

## عن المشروع

موقع شخصي لتدوين الملاحظات والدروس التي أتعلمها في رحلتي كمطور. الهدف منه تتبع تقدمي ومشاركة المعرفة مع المبرمجين العرب، لعلها تنفعهم وتخفف عنهم معاناة البحث.

## المواضيع

يغطي الموقع عدة مجالات:

- **أساسيات** — HTML, JavaScript, ES6, TypeScript, Python, Dart
- **أُطر ومكتبات** — Vue, Nuxt, React, Next.js, React Native, Flutter
- **أدوات** — Git, Neovim, Vite, Prettier, Deno, Sanity, Iconify
- **تصميم** — فن تصميم المواقع وخطواته
- **الأمن السيبراني** — (قريبًا)

## المميزات

- تصميم عربي كامل (RTL) مع خطوط عربية (أميري + تجوال)
- وضع فاتح ومظلم
- تصنيف المحتوى حسب المجال مع فلترة
- مراحل نمو الملاحظة: بذرة 🌱 ، نامية 🌿 ، مكتملة 🌳
- علامة للمحتوى القديم مع اقتراح البديل
- أبيات شعرية في طلب العلم

## التقنيات

- [Astro](https://astro.build) — مولد مواقع ثابتة
- [Astro Content Collections](https://docs.astro.build/en/guides/content-collections/) — إدارة المحتوى بملفات Markdown
- CSS مخصص مع متغيرات للثيمات
- منشور على [Vercel](https://vercel.com)
